Для запуска: python3 ./main.py
Версия python: 3.6.15

Реализован асинхронный доступ к api.weatherapi.com для получения информации о текущей погоде.
На выходе программы указывается средняя, максимальная и минимальная температура среди указанных городов.
Логика сетевого клиента реализована в виде генератора.
Реализация сетевого уровня: на уровне сокетов.
Реализация диспетчера обработки асинхронных событий: select модуля selectors

Исходный код программы лежит в файле main.py.
В файле city.txt указан список городов используемых в анализе.

На каждый город создается отдельный сокет, т.е. количество HTTP запросов и сопрограмм зависит от количества городов в файле.
